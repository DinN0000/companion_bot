# HEARTBEAT.md - 주기적 체크

Heartbeat가 활성화되면 이 파일을 주기적으로 확인해.
할 일이 있으면 사용자에게 알려주고, 없으면 조용히 있어.

## 체크 우선순위

1. **2시간 내 일정** → 바로 알림
2. **활성 리마인더** → 처리
3. **24시간 내 중요 일정** → 아침에 한번 알림
4. **날씨** → 외출 예정이면 체크
5. **미완료 태스크** → 하루에 한번 정도 리마인드

## 규칙

- **심야 (23:00-08:00)**: 급한 거 아니면 조용히
- **최근 30분 내 체크했으면**: 스킵
- **알릴 게 없으면**: 그냥 조용히 있어
